# PW-PRJCT

Course project with automated end-to-end tests built using Playwright.

## Tech Stack

- Playwright
- TypeScript
- Node.js

## Installation

```
npm install
npx playwright install
```

## How to run tests:

```bash
npm run test-login   # runs login tests
npm run test-all     # runs all tests
```

## Test report

npx playwright show-report
